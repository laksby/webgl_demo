## Getting Started

Clone repository

```bash
# Using SSH
git clone git@github.com:laksby/webgl_demo.git

# Using HTTPS
git clone https://github.com/laksby/webgl_demo.git
```

Setup project

```bash
npm install
```

Start project locally

```bash
npm start
```
